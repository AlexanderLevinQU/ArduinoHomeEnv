Arduino project that works with MKR Env SHIELD R2 + MKR WIFI 1010 BOARD to collect sensor data from the env shield and send it to the arduino cloud. 
Must set up your arduino as a thing in the arduino cloud to work. 
Controls smart plug based on humidity. Must change your Sketch Secrets (in arduino cloud) or arduino_secrets.h file for your ip address for your shelly and network credentials.
